# Szakdolgozat – Blokkláncok fejlődése és gyakorlati bemutatása

## Téma rövid leírása

A blokkláncok fejlődésének bemutatása: kezdetektől, a Bitcoin-bányászaton át az okosszerződések elterjedéséig.  
Az elméleti részben áttekintésre kerül:

- a technológia kialakulása,
- főbb mérföldkövek,
- a mögötte álló konszenzusmechanizmusok,
- valamint a blokkláncok jelenlegi és jövőbeli felhasználási területei.

## Programozási rész

A programozási rész a blokklánc-technológia kriptográfiai alapjaira fog épülni, különös tekintettel:

- a hash-elésre,
- a blokkok felépítésére,
- és a láncolás mechanizmusára.

Python nyelven készül egy egyszerű blokklánc-implementáció, amely képes:

- blokkok létrehozására,
- azok összekapcsolására,
- és a lánc érvényességének ellenőrzésére.

A rendszerben bemutatásra kerül, hogyan változik a hash érték bármilyen adatmanipuláció esetén, és opcionálisan beépíthető egy egyszerű proof-of-work algoritmus is, amely szemlélteti a bányászat alapját.

A célom **nem** egy teljes kriptovaluta vagy hálózat szimulálása, hanem a blokklánc működésének alapelveinek gyakorlati bemutatása nagyvonalakban.
